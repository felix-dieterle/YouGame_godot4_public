# Asset Credits Template

Verwende diese Vorlage, um alle verwendeten Assets im Projekt zu dokumentieren. Kopiere den folgenden Block für jedes neue Asset und fülle die Felder aus.

## Template pro Asset

```
### [Asset-Name]

**Pfad im Repo:**  
`pfad/zum/asset/im/projekt`

**Quelle / URL:**  
[Link zur Download-Quelle]

**Autor:**  
[Name des Erstellers/Studios]

**Lizenz:**  
[z.B. CC0, CC-BY 4.0, MIT, etc.]

**Verwendung:**  
[Wo wird das Asset verwendet, z.B. scenes/characters/fox.tscn]

**Hinweise/Notes:**  
[Optionale Anmerkungen, z.B. modifiziert, retexturiert, etc.]

---
```

## Beispiel

```
### Low-Poly Fox Model

**Pfad im Repo:**  
`assets/models/creatures/fox.glb`

**Quelle / URL:**  
https://quaternius.com/packs/ultimateanimals.html

**Autor:**  
Quaternius

**Lizenz:**  
CC0 (Public Domain)

**Verwendung:**  
scenes/characters/fox.tscn

**Hinweise/Notes:**  
Originalmodell ohne Änderungen verwendet.

---
```

## Wichtige Hinweise

- **Immer vor der Nutzung prüfen:** Lizenz und Nutzungsbedingungen jedes Assets überprüfen
- **Attribution:** Bei CC-BY und ähnlichen Lizenzen ist die Namensnennung Pflicht
- **Kommerzielle Nutzung:** Sicherstellen, dass die Lizenz kommerzielle Nutzung erlaubt, falls geplant
- **Modifikationen:** Wenn ein Asset verändert wurde, dies unter "Hinweise/Notes" dokumentieren
- **Links aktuell halten:** Regelmäßig prüfen, ob die Quell-URLs noch gültig sind
